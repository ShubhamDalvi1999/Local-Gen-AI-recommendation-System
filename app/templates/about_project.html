About Classy Books

Project Overview

An AI-powered book recommendation system that combines machine learning with natural language processing.

Technology Stack
  • Python Flask for backend development
  • SQLite database for data storage
  • Ollama for LLM-based chat interactions
  • Vector embeddings for semantic search
  • Tailwind CSS for modern UI design
Key Features
  • Natural language chat interface
  • Personalized book recommendations
  • Semantic search capabilities
  • Detailed book information
  • User-friendly interface
How It Works

The system uses advanced natural language processing to understand user preferences and interests. It combines this understanding with vector embeddings to find semantically similar books in our database. The recommendations are then enhanced using the Ollama LLM to provide natural and contextual responses.
